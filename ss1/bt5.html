Giới thiệu môn Front-end


Frontend là gì

Frontend là phần quan trọng của phát triển web bao gồm mọi thứ mà mọi người có thể nhìn thấy và tương tác trực tiếp với web

Các công nghệ Frontend phổ biến


  • HTML: Ngôn ngữ đánh dấu siêu văn bản là nền tảng để xây dựng cấu trúc trang web
  • CSS: Ngôn ngữ để định dạng và tạo kiểu cho trang web ,giúp trang web trở lên hấp dẫn hơn để dễ sử dụng
  • JaveScript: Ngôn ngữ lập trình giúp cho trang web trở nên động và các tính năng xử lý sự kiện ,hiệu ứng động và kết nối với máy chủ.

Vai trò của Frontend

Frontend giúp tạo ra một trái nghiệm người dùng mượt mà và dễ dàng tiếp cận.Những nhà phát triển Frontend đảm bảo trang web tương thích trên các thiết bị và trinh duyệt khác nhau
